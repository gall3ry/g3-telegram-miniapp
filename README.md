# G3MiniappV2
 
## Ports

Worker: 3100
Backend: 3200

## Development server

Generate types

```bash
pnpm exec nx run-many --target=generate-types
```

## How to start an app

```bash
pnpm nx serve <app-name>
```

Or if you are using nextjs

```bash
pnpm nx dev <app-name>
```

## How to build an app

```bash
pnpm nx build <app-name>
```
